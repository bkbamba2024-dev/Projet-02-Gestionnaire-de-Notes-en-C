# Projet 02 - Gestionnaire de Notes en C

## Description

Ce projet est une application développée en langage C permettant de gérer les notes d'un étudiant à travers un menu interactif.

L'utilisateur peut saisir un ensemble de notes, puis effectuer différentes opérations sans avoir à les ressaisir. Il est également possible de commencer une nouvelle saisie ou de quitter le programme.

Ce projet a été réalisé dans le cadre de mon apprentissage du langage C afin de renforcer mes compétences en programmation avant mon intégration en Génie Informatique.

---

## Fonctionnalités

* Saisie d'un nombre de notes.
* Enregistrement des notes dans un tableau.
* Affichage de toutes les notes.
* Calcul de la moyenne.
* Recherche de la meilleure note.
* Recherche de la plus faible note.
* Nouvelle saisie des notes.
* Quitter le programme.
